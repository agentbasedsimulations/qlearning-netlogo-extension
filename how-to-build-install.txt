Q-learning NetLogo Extension

****************************************************
*** HOW TO OPEN THE PROJECT USING THE ECLIPSE IDE ***
****************************************************
Be sure you have the Eclipse Buildship (Gradle) plugin installed.

1. Clone this repository from Github.
2. On Eclipse, open the following projects:
	- qlearning-netlogo-extension (gradle rootproject)
	- lib (source code project)

The source code is in the 'lib' project.


****************************************************
***		 HOW TO BUILD THE EXTENSION   			 ***
****************************************************
Run the 'jar' gradle task in the 'lib' project to create the 'qlearningextension.jar' file in the 'build/libs/' directory. 

****************************************************
***		 HOW TO INSTALL THE EXTENSION ON NETLOGO ***
****************************************************
1. Open the extensions directory of your NetLogo profile. Usually it is located at:
 - Windows: C:\Users\<your_user>\AppData\Roaming\NetLogo\<netlogo_version_number>\extensions
 - Linux: ~/.netlogo/<netlogo_version_number>/extensions

2. Create directory named 'qlearningextension' within the 'extensions' directory.

3. Copy the 'qlearningextension.jar' to the 'qlearningextension' directory created in step 2.

4. Restart NetLogo, if necessary.
